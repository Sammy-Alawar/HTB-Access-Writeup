## About the Machine
Access is a Windows-based Hack The Box machine that relies heavily on misconfigured services, weak credential storage, and improperly secured file shares. 
The attack path involves FTP enumeration, extracting sensitive data from backups, and leveraging stored credentials to gain Administrator access.

## Overview

This repository contains my full penetration testing writeup for the Hack The Box **Access** machine.

The full PDF writeup in this repository includes screenshots, raw terminal output, and a step-by-step breakdown of the attack path.
